Set up your Security Map

  • Open GoblinPass in a clean browser session first, preferably Incognito/Private Window for first setup testing.
  • Register the YubiKey/passkey that will protect the Security Map.
  • Create your first Security Map entries.
  • Save/export your encrypted GoblinPass State file.
  • Store the exported state file somewhere safe.
  • The exported file cannot be decrypted without the matching YubiKey/passkey.
  • If the YubiKey/passkey is lost, the encrypted state may not be recoverable unless you have another registered recovery method.
