Local Web Tool

Google Backup Codes Vault

Save Google backup codes locally in your browser. New exports are encrypted with a secret produced by your physical YubiKey/security key, so the file can be imported in another supported browser with the same key.

Local only YubiKey PRF unlock Portable encrypted file

Step 1

Set up your YubiKey

Register a PRF-capable physical security key, then use that same key whenever you unlock, save, export, or import your encrypted backup codes.

If Windows shows a passkey picker, choose Security key. This tool is intended for a physical YubiKey/security key, not Windows Hello.

Status: ready. Register a PRF-capable YubiKey first.

Locked Encryption: YubiKey PRF secret Last saved: Never
2FA

Step 2

Encrypted backup codes

Once unlocked, paste or edit your backup codes below. Press save to encrypt them with your YubiKey PRF secret. Exported files include public unlock metadata, but not the YubiKey secret itself.

Important

This is local-only. No backup codes are sent to GoblinPass, Google, or any server.

If you lose the YubiKey/security key used for PRF encryption, GoblinPass cannot recover your codes.

Exported files are encrypted for the same YubiKey, but keep them somewhere private. A fully compromised computer can still read codes after you unlock and display them.
