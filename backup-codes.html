Google Backup Codes Vault

Store Google backup codes locally in your browser, encrypted with a key derived from your YubiKey. No server, no account, no cloud sync.

Local only YubiKey PRF Encrypted storage

Step 1

Set up your YubiKey

This tool uses WebAuthn PRF/hmac-secret. A normal WebAuthn login is not enough, because the YubiKey must return key material for local encryption and decryption.

Status: not set up.

2FA

Step 2

Encrypt backup codes

The codes are encrypted before they are saved to browser local storage.

Step 3

Unlock when needed

To retrieve your codes, connect or tap the same YubiKey and approve the browser prompt. If PRF data is not returned, decryption stops. 

Encrypted backup codes will appear here after YubiKey unlock.

Important

This app cannot recover your backup codes if you lose the YubiKey or the browser storage is cleared. Keep a safe offline backup somewhere you control.

WebAuthn PRF support varies by browser, operating system, and hardware key. GitHub Pages over HTTPS is suitable; opening this page directly from a local file may not allow WebAuthn.

No cryptography here protects against a fully compromised device that can read the screen or capture decrypted codes after unlock.
