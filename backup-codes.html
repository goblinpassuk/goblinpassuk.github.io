Local Web Tool

Google Backup Codes Vault

Save Google backup codes locally in your browser. Unlock, edit, and export them only after signing in with the registered physical YubiKey/security key.

Local only YubiKey unlock Encrypted storage

Step 1

Set up your YubiKey

Register once in this browser, then use the same physical security key whenever you unlock or save your encrypted backup codes.

If Windows shows a passkey picker, choose Security key. This tool is intended for a physical YubiKey/security key, not Windows Hello.

Status: ready. Register your YubiKey first.

Locked Storage: Browser persistent storage Last saved: Never
2FA

Step 2

Encrypted backup codes

Once unlocked, paste or edit your backup codes below. Press save to encrypt them locally into IndexedDB, with localStorage used as a fallback copy.

Step 3

Mark used codes

After using a backup code, tick it below and remove it. The remaining codes stay in the editor until you save again.

Unlock your backup codes to manage individual entries.

Important

This is local-only. No backup codes are sent to GoblinPass, Google, or any server.

If you lose the YubiKey/security key or clear this browser storage, GoblinPass cannot recover your codes.

Exported files are encrypted, but keep them somewhere private. A fully compromised computer can still read codes after you unlock and display them.
